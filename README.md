# ComicVision Sorter - Stage 1

AI-powered comic book metadata extraction and cataloging system.

## Quick Start

**Run the application:**
```
comicvision-sorter-stage1\build\Release\ComicVisionSorterStage1.exe
```

**Full documentation:** See [USER_GUIDE.md](USER_GUIDE.md)

## Features

✅ **Folder Browser with Queue System**
- Browse and select comic book images
- Double-click to add files to processing queue
- Batch process multiple files

✅ **AI-Powered Analysis**
- Support for Ollama and LM Studio
- Vision-capable model integration
- Automatic metadata extraction

✅ **Database Management**
- SQLite database for persistent storage
- Editable results table
- Image viewer with zoom

## Requirements

- Qt 6.7.0+
- Ollama or LM Studio (for AI processing)
- Windows 10/11

## Setup

1. Install Qt 6 and CMake
2. Install Ollama or LM Studio
3. Run the executable
4. Select AI model and start processing
